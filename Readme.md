## BLOG website

### Backend-https://blog-1-9t7q.onrender.com
### Frontend-https://endearing-kheer-c04395.netlify.app/
### Features
* Authentication and Authorization
* user can create a Post
* User can view the all Post
* Valid user only can create a Post

### API Endpoints
* get -> *`/getALLPost`*
* post --> *`/createPost`*
* login --> *`/login`*
* register --> *`/register`*

### To use it
* git clone
* npm install
* add env details 
    * IMAGEKIT_KEY
    * MONGO_URI
    * JWT_SECRET
